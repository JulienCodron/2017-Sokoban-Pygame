Control :

Z Move_up
Q Move_left
S Move_down
D Move_Right

R Restart Lvl
ESCAPE Quit
RETURN Skip lvl

I did this game when i was in 1st year of license. When i was learning programing basics.

This game contain 3 lvl, you have to put all the boxs on the cross. Without blocking them throught the wall.

How to lunch the game:

1 - Install python, you can use idle 3.6. 
2 - Install pyGame library.
3 - Open __main__.py and run it.
